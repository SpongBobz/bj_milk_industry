# bj_milk_industry

## 基础目录结构

```  
|-- WNPMonitorScreen 
	|-- src
	|   |-- assets              //静态资源文件
	|   |-- components          //公共组件
	|   |-- router             //路由
	|   |-- style              //样式文件
	|   |-- utils             
	|   |   |-- cusImportAntd     //按需引入antDV 组件
	|   |-- view               //页面目录
	|   |   |-- components        //弹窗组件
	|   |   |-- home              //首页
	|   |-- main.js           //入口文件   
	|-- vue.config.js         // vue配置文件
```

## 运行环境 
node.js: v10.13.0 
vue-cli: 4.5.3

## 运行

### 安装依赖
```
npm install
```

### 本地启动
```
npm run serve
```

### 打包
```
npm run build
```
